# 課題 02-02 回答

## 問2：何時間働くと再帰的な賃金体系のほうが儲かるか

結論から述べると、**8時間** 働いた時点で再帰的な賃金体系の方が儲かるようになります。

### 1. 比較シミュレーション
時給1,226円の一般賃金と、再帰的賃金の累計額を比較した結果は以下の通りです。

| 勤務時間 | 一般賃金累計 | 再帰的賃金（単価） | 再帰的賃金累計 |
| :--- | :--- | :--- | :--- |
| 1時間 | 1,226円 | 100円 | 100円 |
| 2時間 | 2,452円 | 150円 | 250円 |
| 3時間 | 3,678円 | 250円 | 500円 |
| 4時間 | 4,904円 | 450円 | 950円 |
| 5時間 | 6,130円 | 850円 | 1,800円 |
| 6時間 | 7,356円 | 1,650円 | 3,450円 |
| 7時間 | 8,582円 | 3,250円 | 6,700円 |
| **8時間** | **9,808円** | **6,450円** | **13,150円** |

### 2. 結論
8時間勤務した際、再帰的賃金の累計（13,150円）が一般賃金の累計（9,808円）を上回ります。
よって、8時間以上の勤務であれば再帰的な賃金体系を選択するほうが有利です。
